  ## Installation
  npm install
  npm install express
  npm install yarn
  ## Usage
  Press "Get Started".
  Create a title in "Note Title".
  Create text in "Note Text".
  Press floppy disk in top right hand corner to save note
  Press red trashcan to delete note.
  Press on title you have created to edit or read the note.
